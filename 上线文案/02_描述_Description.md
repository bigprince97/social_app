# Omega — App 描述（Description，≤4000 字符）

---

## 简体中文（zh-Hans）

```
Omega学习是一个经书阅读与学习交流社区。你可以在这里阅读经书内容，标记重点内容，也可以通过动态、评论和消息与朋友交流想法。

【经书阅读】
· 分类浏览经书与学习内容
· 清晰的章节阅读界面
· 支持书签与高亮标记
· 支持引用内容进行分享和讨论

【学习与分享】
· 发布文字、图片、视频与引用内容
· 浏览最新内容、热门内容和话题内容
· 点赞、评论，围绕内容展开交流
· 关注好友，建立自己的学习社区

【消息与小组】
· 一对一私信与群组聊天
· 支持文字、图片、视频、语音、文件等消息形式
· 群组公告、成员管理和管理员权限
· 语音 / 视频通话，方便远程沟通
· 群组直播，多人同屏交流

【友好体验】
· 简体、繁体界面切换
· 消息与来电实时推送，重要内容不错过
· 拉黑与举报功能，共同维护友善、安全的社区环境

Omega学习希望帮助用户更轻松地阅读、标记、分享和交流。无论是个人学习、内容分享，还是小组沟通，都可以在这里完成。
```

· 關注好友，建立自己的學習社區

· 群組公告、成員管理和管理員權限

Omega学习希望幫助使用者更輕鬆地閱讀、標記、分享和交流。無論是個人學習、內容分享，還是小組溝通，都可以在這裡完成。
