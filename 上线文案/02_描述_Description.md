# Omega — App 描述（Description，≤4000 字符）

## 简体中文（zh-Hans）

```
Omega 是一个为信仰而生的经书阅读与社群交流平台。在这里，你可以静心读经、记录灵修，也可以与志同道合的朋友分享见解、彼此连结。

【经书阅读】
· 多信仰经典：基督教圣经、佛经、道家经典等分类阅读
· 章节阅读器：清晰排版，支持阅读进度记录
· 多语言对照：简体、繁体等译本切换
· 书签与高亮：标记喜欢的章节，添加个人笔记
· 经文互引：查看相关经文之间的引用关系
· 离线阅读：章节本地缓存，无网也能读
· 经典音频：边听边读，随时随地默想

【分享与讨论】
· 关注 / 最新 / 热门 / 话题 多种动态流
· 发布文字、图片、视频、语音与经文引用
· 点赞、评论，与同道交流见解
· 关注好友，建立你的信仰社群

【群聊与团契】
· 一对一私信与群组聊天
· 文字、图片、视频、语音、文件多种消息
· 群组公告、成员管理、管理员权限
· 语音 / 视频通话，远程也能面对面团契
· 群组直播，多人同屏交流

【贴心设计】
· 简体、繁体界面随心切换
· 消息与来电实时推送，重要内容不错过
· 拉黑与举报，共同维护清洁友善的社群环境

Omega 致力于打造一个尊重、友善、零容忍不良内容的信仰社群。无论你在世界哪个角落，都能在这里读经、思考、分享，与同道同行。

立即下载 Omega，开启你的灵修与交流之旅。
```
