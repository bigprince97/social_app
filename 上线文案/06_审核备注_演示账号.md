## 备注正文（英文版，审核员主要看英文）

```
Demo account:
Email: __________
Password: __________

About the app:
Omega is a faith community app for reading scriptures (Bible, Buddhist and Taoist texts), with social features (posts, comments, direct & group messaging, and voice/video calls).

How to test key features:
1. Sign in with the demo account above.
2. Scriptures: open the "Scriptures" tab to read chapters, bookmark and highlight.
3. Feed: browse posts, like and comment; create a post from the compose button.
4. Messaging: open "Messages" to start a 1-on-1 or group chat (text, image, voice).
5. Calls: in a chat, tap the call icon to start a voice/video call (uses LiveKit).

User-generated content & moderation:
- Every post, comment and user can be REPORTED via the report option.
- Users can be BLOCKED.
- We have a published EULA with zero-tolerance for objectionable content, and we review/remove reported content and ban offending users within 24 hours.

Permissions: microphone, camera and photo library are requested only when the user sends voice/video messages, makes calls, or attaches media.

Privacy Policy: <你的隐私政策URL>
Contact: bigprince1997@gmail.com
```
