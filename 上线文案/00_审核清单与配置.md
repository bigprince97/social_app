# Omega社区 上线总清单 & App Store Connect 配置答案

> 一站式 checklist。提审前逐项打勾。

## 0. 提审前必须先确认（硬门槛）
- [ ] **举报功能已上线**（动态、评论、消息、用户可举报）
- [ ] 拉黑功能已上线
- [ ] 内容过滤机制（敏感词/人工审核任一）
- [ ] 隐私政策已发布到公开网址
- [ ] EULA 已就绪（App Store Connect 填或网页）
- [ ] App 内有「删除账号」入口（Apple 强制）

## 1. App 基本信息（简体 + 繁体两套）
- [ ] 名称：Omega社区
- [ ] 副标题（见 01 文件）
- [ ] 关键词（见 01 文件）
- [ ] 描述（见 02 文件）
- [ ] 宣传文本（见 01 文件）
- [ ] What's New：首版填「全新上线」

## 2. 必填链接
- [ ] 隐私政策 URL：https://wrcc.one/omega/privacy/
- [ ] 技术支持 URL：https://wrcc.one/omega/support/
- [ ] 营销 URL（选填）

## 3. App 隐私「营养标签」申报（按本 app 实际）
- [ ] 联系信息 → 电子邮箱地址（用于：App 功能 / 账号）
- [ ] 用户内容 → 照片或视频、音频、其他用户内容（用于：App 功能）
- [ ] 标识符 → 用户 ID（用于：App 功能）
- [ ] 诊断 → 崩溃/性能数据（如接入）
- [ ] 是否用于追踪：否（你没有广告/追踪 SDK）

## 4. 年龄分级
- [ ] 按实际填问卷，预计 **17+**
  - 「用户生成内容」→ 是
  - 「不受限制的网页/通讯访问」→ 是（陌生人音视频）
  - 其余成人内容项→ 否

## 5. 价格与销售范围
- [ ] 价格：免费（无 IAP）
- [ ] 销售范围：建议先选中国大陆、香港、台湾、澳门等中文区（你只做简繁）

## 6. 平台 / 设备
- [ ] iPhone only（按你定）→ 仅传 iPhone 截图
- [ ] 截图：6.9"（iPhone 16 Pro Max）+ 6.5" 各一组，每语言一套，3~10 张
- [ ] App 图标 1024×1024 PNG，无透明无圆角

## 7. 出口合规
- [ ] 使用标准 HTTPS 加密 → 通常可声明「符合豁免」（可在 Info.plist 设 ITSAppUsesNonExemptEncryption=NO，此为可选改动）

## 8. App Review Notes（审核备注，给审核员）模板见 06 文件
- [ ] 提供演示账号（邮箱+密码）
- [ ] 说明如何测试通话、如何举报/拉黑

## 9. 构建与上传（技术动作，提审前）
- [ ] Xcode 配置签名（用新注册的开发者账号）
- [ ] Bundle ID：com.zoven.omega（与账号关联，App Store Connect 创建 App 时用）
- [ ] flutter build ipa → 用 Xcode/Transporter 上传（当前 IPA：`build/ios/ipa/Omega.ipa`）
- [ ] 在 App Store Connect 选择该构建版本
